[面试]2020-4-7——腾讯前端一面

[面试]2020-4-9——腾讯前端一面

作者：亚双义、一真

邮箱：chenzhiyuan1999@gmail.com

时间：2020-4-9

前言

其实在10多天前自己投递的是Web开发岗，但前几天却收到了Web前端岗发来的面试邀请。其实原因我也大体知道的：一是我走的内推渠道，简历大概率不会筛掉；二是自己后端技术写的是.net，而现今大厂Web开发语言普遍都是Java，于是我便显得无力；三是自己在简历上写了些许前端技术栈。我当时还在犹豫要不要接受面试邀请，思前想后这不妨是个机会，于是便接受下来，花了2晚自学了JavaScript的知识点以应付面试

过程

让我没想到的是一上来就手撕代码，40分钟内完成6道算法题，说实话这些题除了最后一道手写二叉树之外并不算难，然而自己还是太菜鸡了，堪堪做出2道…后面面试官问了我前端、OS以及计网相关知识，我答出来的也就七八成吧。最后面试官问我还有没有问题，我问了“从现在到暑假还有一段空档期来提升自己，请问您对我有什么建议，或者说我需要注意些什么吗？”，面试官很中肯地说“你现在还是学生，应当从基础做起，比如算法这些，基础做好了再去弄框架这些会更好”。面试结束后我思考了蛮久的，自己确实是平时不注意基础，学校所教课程也只是满足于及格便好。下面直接放出面试题吧，当然有些我忘记或者听不懂的就没录入了

面试题

  1. 算法（6道，40分钟）

    • 洗牌算法：对52张牌洗牌，要求尽量洗乱，而且原牌不能在原位置上重复 void ShuffleArray(int array[], int len)

    • 把字符串类似“12345”转化为10进制数字，不能使用系统函数

    • 数组a[N]，存放了数字1至N-1，其中某个数字重复一次。写一个函数，找出被重复的数字。时间复杂度必须为O(N), 空间复杂度不能是O[N]。 函数原型：

      int find(int a[], int N)

    • 判断正整数是否是对称数，如3， 121， 12321。 不能把整数转为字符串来判断。

      //返回 1：对称； 0： 不对称

      int ismirror(int value)

    • 给定一个整数，按10进制来看，计算里面包含多少个0。 不能用snprintf 转成字符串。

      int count(int value)

    • 有一个二叉树，每个节点的值是一个整数。写一个函数，判断这棵树中是否存在从根到叶子节点的一个路径，这个路径上所有节点之和为某一个值。存在返回1， 否则返回0。

      struct TreeNode

      {

      int value;

      struct TreeNode *left, *right;

      };

      int haspath(struct TreeNode *root, int value)

  2. 闭包概念、好处

  3. 所做项目的流程，特点

  4. JS数据类型

  5. Promise对象

  6. 原型链概念、作用

  7. 缓存

  8. HTTP的概念

  9. HTTPS与HTTP的区别

  10. echarts的作用，如何实现绘制图表功能

  11. git和svn的区别

  12. 要求查看个人博客

  13. 进程和线程区别

  14. 虚拟内存的概念，与物理内存最大的区别是

  15. 局域网中如何实现上传和下载功能（P2P）

总结

自己该放下虚无缥缈的幻想学习了，之前总想着学习Vue、Docker、Redis这些看起来高大上的东西，但其实这些知识其实根本不是现在我这个阶段所能接受的，即使是死记硬背了也完全无法灵活运用，仿佛纸上谈兵一般。

从今天开始要向Java过渡了，原因有蛮多的：.net平台虽然用起来很爽，但是在国内发展得真的不行，一般都是国企才招，而Java岗位遍地都是；这学期学了大数据课程，也是非常热门的领域，其中是要用到Java的；我想把Linux学好，而要应用Linux知识的话.net显得不太实际了，唯有Java能让它发挥一二。除此之外，得好好在牛客网刷算法、基础题了，这次面试的目的确实达到了，挺好的。
